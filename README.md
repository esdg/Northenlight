## Installation

Add the line bellow in your css declarations between head tag.
```
<link href="css/main.css" rel="stylesheet">
```

Add thoses line at the end of the body tag.
```
<script src="js/northtenlight.js"></script>
<script>
  northenlight.init();
</script>
```
